    - > type of project : application
    - > implementation language : JAVA
    - > build script DSL : Groovy
    - > test framework : JUnit 4
- 실습
    - src/main/javaHello2.java    
    - 다른 파일에 있는 파일 컴파일하기
        java -classpath src/main/java Hello2 
        //-classpath 위치를 알려줌
